Rainwater Storage Calculator

Instructions:

The goal of this page is to help calculate an optimum tank size for a given rainwater harvesting sytem. This current version requires an upload of dailyrainfall data. It will only work for a single roof top collection area for a single type of water use. We hope to add a monthly option along with multiple water uses at somepoint as well. Additionally, we hope to store data from rainfall in the Bangalore area, so that it is not neccessary to upload. For now the following is needed:

  1. Daily rainfall data for a year of interest accompanied by the date this was recorded (mm/dd/yyyy format) in an excel file.
  2. The collection area of your roof.
  3. Either the efficieny of the collection or the type of roof selected from a list.
  4. Either information about the population or information about the population and the water demand of the people in this population.
  5. The minimum, maximum, and step size of the storage tanks
  6. *Any potenial days/days of week or reduced water demand

*optional
